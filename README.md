# Getting Started

CRUD on nodeJS.

## Installation

```sh
yarn install
```
## Usage

```sh
yarn dev
```
```sh
routes:
  GET http://localhost:3535/equipments
  POST http://localhost:3535/equipments
  PUT http://localhost:3535/equipments/:id
  DELETE http://localhost:3535/equipments/:id
```

## Testing

```sh
yarn test
```
